LittleLLM - AI Chat Assistant

This software provides a lightweight desktop interface for interacting with
various AI language models including OpenAI, Anthropic, and others.

Features:
- Multi-provider AI chat support
- Global keyboard shortcuts
- System tray integration
- File upload and image support
- Customizable themes and prompts

For more information, visit: https://github.com/NickPittas/littlellm
